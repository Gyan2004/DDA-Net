2. Creates train (800) and validation (200) sets

4. For each epoch:
   - Trains each model on training data
   - Validates on validation data
   - Saves best models automatically
   - Prints progress
5. Stops early if no improvement for 25 epochs
6. Saves checkpoints to checkpoints/ directory
7. Prints final results

After training:
$ python evaluate_models.py

8. Loads all trained models
9. Evaluates on validation set
10. Generates comparison table
11. Creates visualizations:
    - Training curves (loss & dice)
    - Sample predictions
    - Performance comparison
